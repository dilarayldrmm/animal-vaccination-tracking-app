# 🐱 Animal Vaccination Tracking App

A mobile application developed to track cats' vaccination schedules and manage pet information easily.

## 🚀 Features

- Add, edit and delete cat profiles
- Track vaccination dates
- Edit vaccination information
- User authentication system
- Local storage support
- Simple and user-friendly interface

## 🛠️ Technologies Used

- React Native
- Expo
- JavaScript
- Supabase
- AsyncStorage

## 📱 Screens

- Login Screen
- Home Screen
- Add Cat Screen
- Vaccine Tracking Screen
- Edit Screens

## 🎯 Purpose of the Project

This project was developed to help pet owners manage their cats' vaccination schedules more effectively and avoid missing important vaccine dates.
